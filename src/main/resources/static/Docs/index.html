Files Structure

Let's look at the structure of the files:

wicodus/
├── doc/
├── HTML/
		└── assets /
└── sass/

As you can see here there are 3 main directories:

  • doc (Documentation directory)
  • HTML This directory contains the files that you need to get started. All the files you could see in the online demo, including UI elements and templates.

Animate on scroll (aos)

AOS allows you to animate elements as you scroll down, and up. If you scroll back to top, elements will animate to it's previous state and are ready to animate again if you scroll down.

To get a better understanding how this actually works, I encourage you to check this post on CSS-tricks.


Usage

All you have to do is to add data-aos attribute to html element, like so:


<div data-aos="animation_name"></div>

Script will trigger "animation_name" animation on this element, if you scroll to it.

Full information you can find on the Animate on scroll plug-in page.

Owl Carousel

Touch enabled plugin that lets you create a beautiful responsive carousel slider.


<div class="owl-carousel">
	<div class="item"></div>
</div>

Settings

Attribute Description Example value
data-carousel-items The number of items you want to see on the screen. data-carousel-items="1, 3, 5"
(1 - for small devices, 3 - medium, 5 - large.)
data-carousel-margin Margin-right(px) on item. data-carousel-margin="10"
data-carousel-loop Infinity loop. Duplicate last and first items to get loop illusion. data-carousel-loop="true"
data-carousel-center Center item. Works well with even an odd number of items. data-carousel-center="false"
data-carousel-mouseDrag Mouse drag enabled. data-carousel-mouseDrag="true"
data-carousel-touchDrag Touch drag enabled. data-carousel-touchDrag="true"
data-carousel-stagePadding Padding left and right on stage (can see neighbours). data-carousel-stagePadding="0"
data-carousel-autoWidth Set non grid content. Try using width style on divs. data-carousel-autoWidth="false"
data-carousel-startPosition Start position. data-carousel-startPosition="0"
data-carousel-nav Show next/prev buttons. data-carousel-nav="false"
data-carousel-navText Text or HTML code that will be inside the nav buttons. data-carousel-navText="Next, Prev"
data-carousel-navElement DOM element type for a single directional navigation link. data-carousel-navElement="button"
data-carousel-rewind Go backwards when the boundary has reached. data-carousel-rewind="true"
data-carousel-slideBy Navigation slide by x. 'page'. data-carousel-slideBy="1"
data-carousel-slideTransition You can define the transition for the stage you want to use eg. linear. data-carousel-slideTransition="linear"
data-carousel-dots Show dots navigation. data-carousel-dots="true"
data-carousel-lazyLoad Lazy load images. data-src and data-src-retina for highres. Also load images into background inline style if element is not <img>. data-carousel-lazyLoad="false"
data-carousel-autoplay Enable/Disable autoplay. data-carousel-autoplay="true"
data-carousel-autoplayTimeout Autoplay interval timeout. data-carousel-autoplayTimeout="5000"
data-carousel-autoplayHoverPause Pause on mouse hover. data-carousel-autoplayHoverPause="false"
data-carousel-video Enable fetching YouTube/Vimeo/Vzaar videos. data-carousel-video="false"
data-carousel-animateOut Class for CSS3 animation out. data-carousel-animateOut="false"
data-carousel-animateIn Class for CSS3 animation in. data-carousel-animateIn="false"

More info on Official docs page

Scrollbar

Plugin for custom scrollbars on mobile and desktop browsers.


<div data-scrollbar="scroll"></div>

Settings (basic)

Attribute Description Example value
data-scrollbar-color Change cursor color in hex. data-scrollbar-color="true"
data-scrollbar-border Css definition for cursor border. data-scrollbar-border="10"
data-scrollbar-width Cursor width in pixel. data-scrollbar-width="true"
data-scrollbar-zindex Change z-index for scrollbar div. data-scrollbar-zindex="false"

For full customization, set the data-scrollbar="myScrollbar", and add the following code to the end of the document:

$('[data-scrollbar="myScrollbar"]').niceScroll({
  cursorcolor: "#424242",
  cursoropacitymin: 0,
  cursoropacitymax: 1,
  cursorwidth: "5px",
  cursorborder: "1px solid #fff",
  ...
});

Full list of options available on the Official page.
